Projeto Infancia.com

Como dito anteriormente trabalhei como auxiliar de sala nesse projeto chamado de Infancia.com, desenvolvido pela Cidade empreendedora - SEBRAE.
Nesse projeto tive a íncrivel experiência de ajudar/auxiliar professores de instituções públicas e ONGs a desenvolverem habilidades básicas de robótica.
Com intuito de criar uma aula mais dinâmica e interessantes para seus alunos.

Dinâmica do projeto:

Ensinamos através se uma plataforma de "joguinhos" a terem o "pensamento lógico".
Desenvolviamos os códigos em blocos com o auxilio do Pictoblox. Após entenderem a plataforma passavamos para a explicação de alguns conceitos da parte "robótica", como:

  • Polaridade dos componetes;
  • Como funcionam os componentes;
  • Funcionamento do arduino.
Quando estas partes estavam alinhadas, passávamos para o desenvolvimento de um projeto, geralmente um carrinho com programação básica.

Nesse projeto além de aproximar a tecnologia dos alunos, desenvolvemos a parte artistica e a criatividade para resolver problemas.
